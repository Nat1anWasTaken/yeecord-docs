## 邀請機器人

1. 首先，點進 [這個連結](https://yeecord.com/link/)，你應該會進到這個頁面：

  ![邀請機器龍](/img/docs/getting-started/invite-screen.png)

2. 接下來，選擇你想要邀請機器龍到的伺服器，然後按下那個大大的「繼續」按鈕

3. 大功告成！機器龍現在蹦出在你的伺服器裡面了！

![機器龍成功加入](/img/docs/getting-started/bot-joined.png)

## 基本功能

你可以透過 `/help` 來查看機器龍的所有指令：

![幫助指令](/img/docs/getting-started/help-for-all-commands.png)

:::info 有趣的事實
目前，除了幾個功能性指令外，絕大部分的指令都維持在舊有的訊息指令，
我們將會在未來慢慢地將指令全部遷移至斜線指令！
:::

也可以透過 `/help command:{指令}` 來查看特定指令的用法：

![特定幫助指令](/img/docs/getting-started/help-for-specified-command.png)

## 斜線指令的使用訣竅

雖然 Discord 大力推薦斜線指令，
~~但是斜線指令的使用者體驗真的是爛到一個靠北~~，時常出現各種奇奇怪怪的狀況，
在這邊小編統整了幾個使用斜線指令的訣竅，大家可以參考一下！

### 第一點：一定要等待 Auto-Complete

在輸入完指令後，一定要等待 Discord　自動跳出指令，不然只會傳送訊息，~~然後被大家嘲笑~~

![正確示範](/img/docs/getting-started/good-example-of-using-help.gif)

![錯誤示範](/img/docs/getting-started/bad-example-of-using-help.gif)

### 第二點：傳入參數時善用方向鍵

傳入指令參數時，方向鍵一向很有用

![給定參數](/img/docs/getting-started/giving-arguments.gif)
